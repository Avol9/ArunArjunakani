The Power of Choice

Hello. I was considering starting this blog with the phrase "Hello World", as is the case with many CS ventures, but I was never a fan of that. When I was first learning Java I decided to diverge from the book's instructions and I wrote System.out.println("Hello Arun");. I thought I was so cool for doing that. I guess that was one of the first choices I made since I have started learning CS. Ultimately an irrelavant choice, but one I don't regret making. Since then, I've never followed a tutorial that's told me to print out "Hello World" and I've always replaced it with my own fun string.
